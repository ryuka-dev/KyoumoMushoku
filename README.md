# 今日も無職。

拡大し続ける都市の中で、無職の主人公が拾い集め・単発バイト・買い物・警察からの逃走によって生き延びる、2D 横スクロール型サバイバル探索ゲーム。

水一本の購入を真剣に悩む日々から始まり、やがて都市の資源生態系を制するまで──ブラックユーモアと社会風刺を効かせた小規模作品です。

## ダウンロード

[Releases ページ](https://github.com/ryuka-dev/KyoumoMushoku/releases) から最新版を入手できます（Windowsのみ）。

## 操作

| キー | 動作 |
| --- | --- |
| A / D、← / → | 移動 |
| Shift | 走る |
| E  | 調べる・拾う・話す |
| 数字キー | メニューの選択 |
| Esc | ポーズ・設定・戻る |
| F1 | Debug stats |
